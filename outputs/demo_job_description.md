# Demo Job Description

## Python Backend Developer - AI Recruitment Tools

We are hiring a Python backend developer to help build AI-powered recruiting tools for HR teams.

The ideal candidate should have experience with:

- Python
- FastAPI
- SQL or SQLite
- Docker
- Git
- Machine Learning basics
- REST API development

Nice to have:

- Streamlit dashboards
- React frontend knowledge
- Experience building internal tools
- Cloud deployment experience

The role involves parsing resume data, ranking candidates, building API endpoints, and creating simple dashboards for hiring teams.
